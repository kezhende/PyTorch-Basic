# PyTorch-Basic

PyTorch社区活跃人物陈云编写的《深度学习框架PyTorch入门与实践》的学习代码记录。
 
 
在2018年寒假伊始，在京东和淘宝上发现了国内第二本关于PyTorch的教程书（第一本是廖星宇的《深度学习入门之PyTorch》）。
 
 
初步印象，印刷比较好，排版工整，代码教程比较人性化，感觉良好。
